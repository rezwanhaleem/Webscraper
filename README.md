# Description

Simple web scraping Java app. This app utilizes [Java AWT](https://www.javatpoint.com/java-awt) (Abstract Window Toolkit) for its GUI, [Apache POI](https://poi.apache.org/) for manipulating Microsoft Excel (.xls,.xlsx) Spread Sheets and [Jsoup](https://jsoup.org/) for parsing HTML pulled from links. 

The app is personalized, as it was used for a freelancer project to scrape information on particular websites, regarding non-related Google Ads.

# Usage

Requires Java to be installed before hand.

Run the file `Webscraper.jar` file in /app folder.
