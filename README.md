# Webscraper
Simple web scraping Java app that utilizes Java AWT (Abstract Window Toolkit) for its GUI, Apache POI for manipulating Microsoft Excel ( .xls, .xlsx) Spread Sheets, Google Custom Search API and Jsoup for parsing HTML pulled from links.

<img src="https://i.imgur.com/w5kHHS6.png" alt="drawing" style="width:600px;"/>

<br/>

# Description
Webscraper tool was created to aid in a freelance project for scraping the websites Techyv.com and Nettv4u.com. Project responsibilities consisted of the following:

* Search unique keywords in Google daily from an Excel file
* Arrived at either Techyv or Nettv4u article pages via the search result 
* Identify any unrelated adverts on the site
* Record the advert details along with site screenshots
* Compile the session info into Docx files ready for upload

Following libraries and Integrations were used:
<pre>
Maven
Java Swing 
Java AWT
Apache POI
Jsoup
Dotenv
Google Custom Search API
File Explorer
Excel and Doc Files (.xlsx and .docx)
Org JSON
Java.NET
Multithreading
JavaFX built-in web brower
</pre>
